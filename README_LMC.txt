
# README for Loyalty Membership Coin (LMC)

---

## **Deutsch**

### **Überblick**
Willkommen beim LMC-Projekt! Dieses Repository enthält die wichtigsten Dateien für die Nutzung des Desktop-Wallets und des Miners. Mit LMC bieten wir eine innovative Kryptowährung basierend auf dem Proof-of-Responsibility (PoR) Algorithmus.

---

### **Inhalt**
1. `Desktop_LMC_Wallet.py`: Wallet-Programm zur Verwaltung von LMC.
2. `LMC_Miner.py`: Software für das Mining von LMC.
3. `balance.hash` und `balance.json`: Speichert die aktuellen Wallet-Guthaben und deren Integrität.

---

### **Anleitung**

#### **1. Voraussetzungen**
- **Python-Version:** 3.8 oder höher.
- Abhängigkeiten:
  ```bash
  pip install tkinter hashlib
  ```

#### **2. Nutzung des Wallets**
1. Führe `Desktop_LMC_Wallet.py` aus.
2. Funktionen:
   - **Wallet erstellen:** Erstellt eine neue Wallet-Adresse.
   - **Guthaben anzeigen:** Zeigt das aktuelle Guthaben der Wallet an.
   - **Transaktion senden:** Sende LMC an eine andere Wallet-Adresse.

#### **3. Nutzung des Miners**
1. Führe `LMC_Miner.py` aus.
2. Mining beginnt automatisch. Das System überprüft Hashes und belohnt erfolgreich abgeschlossene Blöcke.

#### **4. Sicherheit**
- Die Datei `balance.hash` stellt sicher, dass die Wallet-Daten nicht manipuliert wurden.
- Ändere keine Dateien direkt, um Integritätsprobleme zu vermeiden.

## **English**

### **Overview**
Welcome to the LMC Project! This repository contains the essential files for using the desktop wallet and miner. LMC offers an innovative cryptocurrency based on the Proof-of-Responsibility (PoR) algorithm.

---

### **Contents**
1. `Desktop_LMC_Wallet.py`: Wallet application for managing LMC.
2. `LMC_Miner.py`: Software for mining LMC.
3. `balance.hash` and `balance.json`: Stores current wallet balances and ensures data integrity.

---

### **Instructions**

#### **1. Requirements**
- **Python Version:** 3.8 or higher.
- Dependencies:
  ```bash
  pip install tkinter hashlib
  ```

#### **2. Using the Wallet**
1. Run `Desktop_LMC_Wallet.py`.
2. Features:
   - **Create Wallet:** Generates a new wallet address.
   - **View Balance:** Displays the current wallet balance.
   - **Send Transaction:** Send LMC to another wallet address.

#### **3. Using the Miner**
1. Run `LMC_Miner.py`.
2. Mining starts automatically. The system verifies hashes and rewards successfully completed blocks.

#### **4. Security**
- The file `balance.hash` ensures that wallet data has not been tampered with.
- Do not modify files directly to avoid integrity issues.
